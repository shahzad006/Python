##  2Ô∏è‚É£Arithmetic Operators
- ‚ûï `+` Add ‚Üí `5 + 3 = 8`  
- ‚ûñ `-` Subtract ‚Üí `5 - 3 = 2`  
- ‚úñÔ∏è `*` Multiply ‚Üí `5 * 3 = 15`  
- ‚ûó `/` Division ‚Üí `10 / 3 = 3.33` (float)  
- ü™ô `%` Modulus ‚Üí `10 % 3 = 1` (remainder)  
- üîº `**` Exponent ‚Üí `2 ** 3 = 8`  
- üßÆ `//` Floor Division ‚Üí `10 // 3 = 3` (no decimals)  

üí° **Difference:** `/` gives float, `//` gives integer.

## 3Ô∏è‚É£ Comparison Operators
- > Greater ‚Üí `5 > 3 ‚úÖ`
- < Less ‚Üí `5 < 3 ‚ùå`
- == Equal ‚Üí `5 == 5 ‚úÖ`
- != Not Equal ‚Üí `5 != 3 ‚úÖ`
- >= Greater/Equal
- <= Less/Equal

```python

print(x > y)   # False
print(x != y)  # True
```
---

## 4Ô∏è‚É£ Logical Operators

- and ‚Üí True if both true
- or ‚Üí True if at least one true
- not ‚Üí Reverses value
```python
a, b = True, False
print(a and b)  # False
print(a or b)   # True
print(not a)    # False
```
